Unsupervised learning solutions

General questions

What is clustering?

Clustering is way to classify different elements into similar groups. Like separating children in boys and girls, but with elements with many variables and many groups. This is known as unsupervised machine learning, and can help to differentiate a patient's cancerous cells from healthy cells


Cool. Can I try the web before signing up?

Yes! Click here to start a simple demo and learn how it works.

There you can use 2 diferent preloaded dataset to play with the different options the web offers you


Which Algorithms can I use?

Right now there are 2 differents algorithms: K-Means and Hierachical clustering. They are different and can give different results. They also accept different parametres, so the results can be quite different according to the data. Those are the most common algorithms used on clustering.


Which of those 2 algorithms is better for my project X?

Any clustering method has a different mathematical basis and thus can give different results once you click the run button. Because of that, there is no best algoithm, each dataset might fit better to one of those. So feel free to try different combinations of algorithms and parameters to find which best fits your data

Groups

What are groups?

A group is a community of different users that can share different projects, results and dataset. Whenever you create a project you can choose to which of the groups you belong is assigned


But what if I don't want to share a project?

You will always have that option, even if you don't belong to any group. Just select the option 'Private' when you start a new project (which is the default option)


How do I add my colleagues to the project

In the group settings, you can search for your colleagues, and find them with their username or password. However, you need admin rights in this group to be able to add people.


Wait, then how I become an admin of a group?

The creator of a group is automatically the first admin of the group. Only a existing admin can select other users to give them admin powers (Remember that with great power comes great responsibility).

What can I do as an admin?

If you are the admin of a group, besides adding new members and making new adminds, you can delete members and configure the projects of the group in the project page: change description, delete the project...


About the dataset

What input datasets can I use?

You need to use a tabulated plain text file: csv-like (comma separated values) files. They can be separated by commas (,) tabs or semicolons (;). By default, the input shows comma separated but it can be changed before create the new project. Here there is an example, which is separated by tabs. If the csv has headers or row identifiers, you have to explicitly indicate it.


About the result

What results do I get

In the result page you can see:

  • The input parameters you used
  • A plot summarizing the clustering and showing the different groups
  • A table with the raw result data
  • An option to compare with other runs

Can I recover previous runs of a project?

Of course! Clustomics saves every previous run in the project page. There you can check those runs and compare them.

Pro tip: The ID and the URL of every run is uniq, so you can save special runs as a bookmar easily!
